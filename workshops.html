Workshop

Mobile app development with JavaScript and React Native

Saturday 21st July

Have you ever wanted to develop a mobile app purely in JavaScript? Fed up with sluggish experience of hybrid apps? With React Native you can now get a native mobile app written fully in JS. The best is yet to come. You write code once for both iPhone and Android. Even on your old Linux laptop!

This will be very hands on workshop and will cover the following topics:

  • How to setup a new project in React Native
  • Expo - what it is, how it works and its limitations
  • When do you need to eject from Expo
  • Navigation options available
  • Debugging React Native applications
  • How to do animations
  • App shipping to Google Play and Apple Store
  • Code maintenance, updates and upgrades
